# # Joh Mustad Tubon (1966) Emulation

A Cmajor DSP image of the rare tube‑shaped, strap‑on monophonic bass organ / proto‑keytar used by Paul McCartney, Ralf Hütter (Kraftwerk), and Scandinavian pop bands.

Memory anchor:
> “No watermarks — the bon bon is the toy synthesizer that the Beatles used. I was emulating it but don’t remember. Clue: that is all they use.”
